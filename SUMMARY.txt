=================================================================
TROUBLESHOOTING UMUM
=================================================================

❌ "Connection Error" saat check NISN
✓ Solusi:
  - Pastikan MySQL sudah running (start service)
  - Cek credentials di config/Database.php
  - Cek database sudah dibuat dan schema di-import

❌ File upload gagal
✓ Solusi:
  - Cek permission folder uploads/ijazah/ (chmod 755)
  - Verifikasi file < 1MB
  - Cek format (hanya JPG, JPEG, PNG)

❌ NISN tidak ditemukan (error 404)
✓ Solusi:
  - Insert sample data terlebih dahulu
  - Gunakan form add-siswa.html untuk input
  - Atau insert manual via query SQL

❌ Halaman admin kosong (tidak ada data)
✓ Solusi:
  - Klik tombol "CARI" untuk load data
  - Pastikan ada data di database
  - Cek browser console untuk JS errors

❌ White screen (error 500)
✓ Solusi:
  - Enable display_errors di php.ini untuk debug
  - Check PHP error logs
  - Verifikasi mysqli extension enabled
  - Test koneksi database dengan simple script
