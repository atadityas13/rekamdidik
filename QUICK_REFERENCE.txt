┌──────────────────────────────────────────────────────────────────────────┐
│ 📞 SUPPORT & TROUBLESHOOTING                                             │
└──────────────────────────────────────────────────────────────────────────┘

COMMON ISSUES:

❌ "Connection Error"
   → Check MySQL is running
   → Verify credentials in config/Database.php

❌ "NISN not found"
   → Insert sample data first
   → Use form at add-siswa.html

❌ "Upload failed"
   → Check folder permission (chmod 755)
   → Verify file size < 1MB
   → Check file format (JPG/JPEG/PNG)

❌ "No data in admin"
   → Click "Cari" button to load
   → Check database connection
   → Check developer console (F12)

❌ "White screen / Error 500"
   → Enable display_errors in php.ini
   → Check PHP error logs
   → Verify mysqli extension enabled

For more help → See README.md & SETUP_GUIDE.md
